# mSearch Web UI

mSearch的Web用户界面，基于Vue 3和Element Plus构建。

## 开发环境搭建

1. 确保已安装Node.js (推荐版本 16+)
2. 安装依赖：
   ```bash
   npm install
   ```

## 开发

启动开发服务器：
```bash
npm run dev
```

## 构建

构建生产版本：
```bash
npm run build
```

构建后的文件将输出到 `../src/gui/dist` 目录。

## 技术栈

- Vue 3
- Element Plus
- Vite
- Axios
